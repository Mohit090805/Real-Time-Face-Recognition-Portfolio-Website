# Real-Time-Face-Recognition-Portfolio-Website
Project Overview Full-stack web application demonstrating real-time face recognition using computer vision and machine learning. Captures live video from a webcam and detects faces in real time. Identifies individuals by comparing detected faces with a stored face database.Applications Smart attendance systems.Surveillance and monitoring systems.
